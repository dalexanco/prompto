## Development

<details>
<summary>
Pre-requisites
</summary> <br />
To be able to start development on amplication make sure that you have the following pre-requisites installed:

###

- Node.js v18 or above
- Git

Then install the project dependencies :

```bash
pnpm install
```

</details>

<details>
<summary>
Scripts
</summary> <br />

### Local development

Then run the development server

```bash
pnpm run dev
```

You can now load the "unpacked" extension from the folder `dist/`.<br/>
The extension is now added to your browser.

### Lint

```bash
pnpm run lint
```

### Build

```bash
pnpm run build
```

### Test

```bash
pnpm run test
```

View and interact with your tests via UI.

```bash
pnpm run test:ui
```

</details>
